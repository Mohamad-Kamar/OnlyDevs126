# Only Devs #126 — Systems Thinking for Software Developers

Talk materials for **Systems Thinking for Software Developers**, presented at [Only Devs #126](https://www.meetup.com/only_devs_bangkok/events/315553743/) in Bangkok on 16 July 2026.

The central practice is simple:

> Do not stop at the component. Keep following the effect.

The talk follows the chain **Code → Delivery → AI → Product → People** and ends with one question: *What is the system teaching people to do?*
